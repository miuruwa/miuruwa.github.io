# Шаблон приложений с использованием WebXUI компонентов
## Полезные ссылки
* [Использовать шаблон](https://github.com/webx-ui/template/generate)

## Полезные команды
* Установить все нужные зависимости (NPM)
```bash
npm i
```

* Запустить приложение (NPM)
```bash
npm start
```

* Деплой на Github Pages
```bash
npm run deploy
```
